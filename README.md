# Attendance Dashboard - Power BI

## Project Overview
This project is an interactive Attendance Dashboard built using Power BI to monitor and analyze attendance data.

## Features
- Attendance KPI Cards
- Attendance Trend Analysis
- Department-wise Attendance Tracking
- Interactive Filters and Slicers
- Data Cleaning and Transformation
- Dynamic Visualizations

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key Insights
- Monitored attendance performance over time
- Identified attendance patterns and trends
- Improved reporting through interactive visualizations
